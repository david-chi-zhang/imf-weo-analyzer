# IMF WEO Analyzer - 更新日志

## [1.1.0] - 2026-03-24

### 🎉 新增功能

#### 数据表格展示
- ✨ 图表下方自动显示完整数据表格
- ✨ 显示所有年份和国家的精确数值
- ✨ 表格支持横向和纵向滚动（max-height: 400px）
- ✨ 鼠标悬停行高亮效果

#### 视觉优化
- ✨ 预测年份（2025+）用黄色背景标记
- ✨ 预测年份添加 * 号标识
- ✨ 加权平均列用粉色背景突出显示
- ✨ 表头固定（sticky positioning）

#### 数据导出
- ✨ 新增 CSV 下载按钮
- ✨ CSV 支持 UTF-8 BOM 编码（Excel 兼容）
- ✨ CSV 包含所有图表数据
- ✨ 自动文件命名（基于图表标题）

### 📝 技术改进

#### chart.js
- 新增 `generateDataTable()` 函数生成表格 HTML
- 优化 CSS 样式，增加表格相关样式
- 新增 `downloadCSV()` JavaScript 函数
- 数据表格自动嵌入 HTML 输出

#### 样式优化
- 表格响应式设计
- 预测行特殊样式（`.forecast-row`）
- 加权列特殊样式（`.weighted-col`）
- 滚动区域优化（`.table-wrapper`）

### 📊 示例

**生成带表格的图表**:
```bash
node src/index.js "金砖五国 2010-2030 年 政府债务"
```

**输出**:
- 交互式折线图
- 完整数据表格（11 行 × 7 列）
- CSV 下载按钮
- PNG 下载按钮

**示例文件**: `output/brics_debt_with_table.html`

### 📈 效果对比

| 版本 | 文件大小 | 数据表格 | CSV 下载 | 预测标记 |
|------|---------|---------|---------|---------|
| v1.0.0 | ~7KB | ❌ | ❌ | 灰色背景 |
| v1.1.0 | ~15KB | ✅ | ✅ | 黄色背景 + * 号 |

## [1.0.0] - 2026-03-24

### 🚀 首发版本

#### 核心功能
- ✨ 数据查询（国家 × 指标 × 时间）
- ✨ 计算模块（增速、指数化、平均、加权平均）
- ✨ 可视化（ECharts 交互式图表）
- ✨ 指标定义查询
- ✨ 国家备注查询
- ✨ 列表查询（指标/国家）

#### 计算功能
- ✨ 同比增速计算
- ✨ 指数化（基期=100）
- ✨ CAGR（复合年增长率）
- ✨ 算术平均/几何平均
- ✨ 分段计算
- ✨ 跨国加权平均（PPP GDP 权重）

#### 可视化
- ✨ 折线图（支持多国对比）
- ✨ 柱状图
- ✨ 多轴图表
- ✨ 预测区域标记
- ✨ 图例交互
- ✨ PNG 下载

#### 数据优化
- ✨ 预建索引（15KB vs 50KB CSV）
- ✨ 指标定义缓存
- ✨ 国家备注缓存
- ✨ 指标别名映射（56 个）
- ✨ 国家组别名映射（53 个）
- ✨ Token 节约 85%+

#### 覆盖范围
- 📊 210 个国家/地区
- 📊 145 个经济指标
- 📊 1980-2030 年时间序列
- 📊 8200 个时间序列

### 📦 技术栈
- Node.js >= 14.0.0
- ECharts 5.4.3
- csv-parser 3.0.0
