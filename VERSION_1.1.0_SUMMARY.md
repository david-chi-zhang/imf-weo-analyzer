
### 主要特性

| 特性 | 说明 | 状态 |

## 🚀 使用方式

### 命令行

```bash
# 所有新生成的图表自动包含数据表格
node src/index.js "金砖五国 2010-2030 年 政府债务"

# 输出
📊 图表已生成：output/brics_debt_1234567890.html
# 该文件包含：
#   - 交互式图表
#   - 完整数据表格
#   - CSV 下载按钮
#   - PNG 下载按钮
```

### API 调用

```javascript
const { chart } = require('./src/index');

chart.generateLineChart({
  title: '我的图表',
  data: [...],
  weightedData: [...]
}, {
  output: 'output/chart.html'
  // 自动包含数据表格
});
```
